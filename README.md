# Expression_Evaluator
A simple expression evaluator written in C.

## Features
- Basic arithmetic: `+`, `-`, `*`, `/`
- Operator precedence: `2 + 3 * 4 = 14`
- Parentheses: `(2 + 3) * 4 = 20`
- Multi-digit numbers: `111 + 143 = 254`

## Build & Run
gcc main.c -o main && ./main

## Usage
> 2 + 3
5
> (12 + 2) * 3
42
> exit
